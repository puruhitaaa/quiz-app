## Features

- **React 18**: Leverage the latest features and improvements of React.
- **TypeScript**: Enjoy type safety and better developer experience.
- **Vite**: Lightning-fast development server and build tool.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **Radix UI**: Accessible and unstyled UI components.
- **React Query**: Powerful data fetching and caching.
- **Supabase**: Seamless integration with Supabase for backend services.
- **Zod**: Schema validation and parsing.
- **Hook Form**: Simple and flexible forms with validation.

## Getting Started

To get started with the Quiz App, follow these steps:

1. **Clone the repository:**

   ```sh
   git clone https://github.com/puruhitaaa/quiz-app.git
   cd quiz-app
   ```

2. **Install dependencies:**

   ```sh
   npm install
   ```

3. **Start the development server:**

   ```sh
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` to see the application in action.

## Scripts

- **`dev`**: Starts the development server using Vite.
- **`build`**: Compiles TypeScript and builds the project using Vite.
- **`lint`**: Lints the code using Biome.
- **`preview`**: Previews the built project using Vite.
